## Madagascar Project

1. Install pipenv

   ```
   pip install pipenv
   (python3.11 --version)
   ```

2. Create a Virtual Environment with python version 3.11

   ```
   pipenv --python 3.11
   ```

3. Install packages:

   ```
   pipenv install
   pipenv install django==4.2
   ```

4. Activate the Virtual Environment:

   - Windows:

   ```
   pipenv shell
   ```

   - macOS/Linux:

   ```
   pipenv shell
   ```

5. Run project:

   ```
   ./manage.py runserver
   ```
